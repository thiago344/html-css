Pré-formatação:

        O codigo abaixo esta escrito na linguagem Java:
          
    public static void main(String []args) {
        Scanner leia = new Scanner(System.in);
        double saldo, reajuste, total;
        System.out.println("Informe seu saldo e nós iremos calcular o reajuste: ");
        reajuste = 0.01 + 1;
        saldo = leia.nextDouble();
        total = (saldo*reajuste);
        System.out.println("O seu saldo com o reajuste é de: " + total);
        leia.close();
    }
